# .banners

Similar to avatars.rotur.dev/username, but instead of the profile picture, its the banner.

### About

* Images are all stored as jpeg
* Images are 900x300
* If no banner is found, you get a 3x1 transparent image

### How do i get a banner?

Same as how you would get a profile picture, but adding .banners:

* https://avatars.rotur.dev/.banners/mist
* https://avatars.rotur.dev/.banners/MiSt

### Query Parameters

* `radius`: Specify the border radius of the banner. Default is 0 (square). Example: `?radius=128`
* `no_animate`: Set to `1` to force an animated (GIF) banner to be served as a still image. The value must be exactly `1` — any other value is ignored. Example: `?no_animate=1`
